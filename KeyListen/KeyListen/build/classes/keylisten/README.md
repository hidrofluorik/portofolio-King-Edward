# PROJECT-KEY-LISTEN
KeyListen adalah pemutar musik berbasis Java dengan antarmuka sederhana yang memudahkan pemutaran file MP3 dari sistem. Selain hiburan, aplikasi ini menjadi sarana belajar Java, khususnya file handling, GUI, dan library audio, serta mendukung pengembangan fitur seperti play/pause, volume, dan daftar favorit.
